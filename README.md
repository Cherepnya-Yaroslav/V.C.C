# V.C.C

V.C.C. это концептуальный fashion web-проект, сделанный как цифровой бренд-экспириенс, а не как обычный интернет-магазин.

Сайт: [viborgclub.fun](https://viborgclub.fun)

## Что это за проект

Это frontend-кейс для бренда одежды в эстетике editorial streetwear.

Задача проекта:
- собрать сильный визуальный образ бренда
- сделать mobile-first интерфейс
- показать одежду как часть атмосферы и культурного контекста
- уйти от типичных ecommerce-паттернов вроде фильтров, корзины и checkout

Идея в том, что V.C.C показывает не “витрину магазина”, а curated fashion experience: одежду, настроение, ритм, визуальный код своей тусовки.

## Что реализовано

- интро-экран с брендовой подачей
- отдельный загрузочный экран перед входом
- каталог в editorial-стилистике
- страницы отдельных товаров
- анимированные переходы между страницами
- собственная визуальная система: типографика, цвет, текстуры, motion
- адаптивная mobile-first верстка
- базовый SEO-набор: metadata, `robots.txt`, `sitemap.xml`, favicon
- production-деплой на VPS с `nginx` и `https`

## Основные страницы

### `/`
Интро-экран с брендовой заставкой, логотипом, motion-эффектами и входом в каталог.

### `/catalog`
Каталог с акцентом на композицию, типографику и атмосферу, а не на marketplace-механику.

### `/product/[slug]`
Страница товара с image-first подачей, описанием и ценой.

## Как сейчас устроены данные

На текущем этапе товары хранятся в локальном mock-data слое. Это сделано специально, чтобы быстро развивать и шлифовать frontend-часть без лишней сложности.

При необходимости проект можно подключить к CMS на следующем этапе.

## Что уже сделано по производительности

- убран transition с самого первого захода на сайт
- изображения товаров переведены в `webp`
- загрузка изображений упрощена для мобильных устройств
- публичные страницы собираются статически

## Запуск локально

```bash
npm install
npm run dev
```

После этого проект будет доступен на `http://localhost:3000`.

## Задача проекта в одном абзаце

V.C.C. это пример того, как сайт бренда одежды может ощущаться не как каталог товаров, а как digital campaign experience. Это кейс про креативный frontend, визуальную выразительность, продуктовый подход и умение доводить идею до живого production-результата.
